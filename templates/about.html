Methodology & Architecture

A comprehensive overview of how SmartGuard bridges the gap between traditional rule-based filtering and stochastic machine learning models.

Structural Features

SmartGuard does not rely on simple string matching. It extracts a 14-dimensional feature vector from every request, analyzing:

  • Token Entropy
  • SQL Keyword Density
  • Special Char Ratio

Hybrid Training

The core Random Forest model was trained on a curated mix of the CSIC 2010 HTTP Dataset and custom OWASP injection payloads.
